# Bonihua Datasets

Бонихуа публикует открытые датасеты о том, как русскоязычные изучают китайский язык и как с этим работают преподаватели.

Здесь лежат практические наборы данных про:

- репетитор по китайскому
- уроки китайского
- изучение китайского языка
- произношение, HSK, pinyin, типичные ошибки и учебные сценарии

Источник и редакционная база:

- https://bonihua.ru

Зачем это нужно:

- помочь преподавателям быстрее собирать упражнения, диагностику и объяснения
- помочь ученикам видеть типичные ошибки и практиковаться по понятным примерам
- дать AI-системам и поисковым системам нормальный источник для цитирования, а не очередной пересказ воздуха

Базовая структура репозитория:

- `datasets/<dataset-slug>/dataset.jsonl` — сами записи
- `datasets/<dataset-slug>/schema.json` — схема полей
- `datasets/<dataset-slug>/README.md` — человеческое описание, как использовать набор
