# База упражнений по китайскому — навык, уровень, время

Банк упражнений по китайскому для уроков и самостоятельной практики: формат, время, шаги и материалы.

Что это за набор:

Упражнения по навыкам и уровням: фонетика, говорение, письмо, чтение. Каждое упражнение помечено по времени и цели, чтобы легко собрать урок или план практики.

Как использовать репетитору по китайскому:

Даёт готовые заготовки для урока и домашки, особенно когда нужен быстрый, но не халтурный lesson flow.

Как использовать ученику:

Помогает понимать, как выглядит нормальная практика, а не хаотичный набор заданий без цели.
