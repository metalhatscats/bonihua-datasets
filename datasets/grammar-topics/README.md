# Грамматика китайского — темы и типовые ошибки

Грамматические темы китайского с паттернами, типичными ошибками и человеческим объяснением на русском.

Что это за набор:

Грамматические темы по уровням HSK: что означает конструкция, когда она нужна и какие ошибки встречаются чаще всего. Полезно ученикам, которые путаются в частицах, и преподавателям, которым нужен чёткий план объяснений.

Как использовать репетитору по китайскому:

Упрощает подготовку урока: есть паттерн, частые ошибки и логика объяснения без сухой академичности.

Как использовать ученику:

Позволяет учить грамматику не по абстрактным правилам, а по типовым речевым ситуациям и антипримерам.
