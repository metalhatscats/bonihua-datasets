# HSK словарь по уровням — тематические наборы

Тематические словарные наборы HSK, пригодные для уроков, повторения и сборки разговорных блоков.

Что это за набор:

Срезы словаря HSK по уровням и темам: что реально нужно для диалогов, чтения и экзамена. Наборы помогают учить слова в контексте, а не бессвязным списком, и быстрее выходить на практику.

Как использовать репетитору по китайскому:

Можно быстро собирать словарь под тему урока, мини-диалоги и повторение по уровням.

Как использовать ученику:

Помогает учить словарь HSK не россыпью, а нормальными смысловыми кластерами.
