# Шаблоны диалогов на китайском — по уровням

Диалоговые шаблоны для китайского: бытовые, учебные и речевые сценарии, которые можно сразу ставить в урок.

Что это за набор:

Готовые сценарии диалогов: знакомство, заказ, перенос встречи. Помогают быстро говорить по делу, а не строить фразу в голове каждую минуту.

Как использовать репетитору по китайскому:

Экономит время на построение разговорной практики: есть готовый каркас, который легко адаптировать под уровень.

Как использовать ученику:

Даёт нормальную разговорную опору вместо бесконечного заучивания слов без ситуации.
