# Типичные ошибки русскоязычных — китайский без ловушек

Типичные ошибки русскоязычных в китайском: лексика, грамматика, порядок слов и исправления без канцелярита.

Что это за набор:

Список ошибок, которые чаще всего совершают русскоязычные: тоны, порядок слов, счётные слова. Каждая карточка показывает, почему ошибка возникает и как её исправить на практике.

Как использовать репетитору по китайскому:

Можно быстро собирать урок вокруг конкретной ошибки: где она всплывает, как объяснить и чем закрепить.

Как использовать ученику:

Удобно использовать как журнал ошибок: находить свои промахи и переводить их в короткие практические действия.
