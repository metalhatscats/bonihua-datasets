# Шаблоны планов уроков китайского

Шаблоны планов уроков по китайскому: структура, тайминг, фокус и логика занятия.

Что это за набор:

Готовые планы уроков по уровням и целям: структура, блоки, длительность, фокус навыков. Помогают преподавателю собрать урок без лишних импровизаций и держать результат.

Как использовать репетитору по китайскому:

Ускоряет подготовку и помогает не скатываться в импровизацию, когда нужен понятный каркас урока.

Как использовать ученику:

Показывает, как должен выглядеть внятный урок китайского, если преподаватель работает системно.
