# Произношение китайского — ловушки и пары звуков

Ловушки произношения китайского для русскоязычных: тоны, pinyin, финали, согласные и ритм короткой фразы.

Что это за набор:

Проблемные пары звуков и тоновые ловушки, из‑за которых речь становится непонятной. Карточки помогают услышать разницу, поставить артикуляцию и закрепить её в минимальных парах.

Как использовать репетитору по китайскому:

Подходит для фонетической диагностики, коротких домашек и объяснения того, почему ученик говорит почти правильно, но звучит мимо смысла.

Как использовать ученику:

Помогает брать одну фонетическую проблему за раз, отрабатывать минимальные пары и собирать привычку короткой записи с самопроверкой.
