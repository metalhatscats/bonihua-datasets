# Методики обучения китайскому — что работает

Методики изучения китайского: что реально работает для речи, аудирования, фонетики и удержания слов.

Что это за набор:

Практические методики обучения: shadowing, интервальные повторения, фразовые блоки и микро‑дриллы. Каждая карточка объясняет, кому метод подходит, как его внедрять и какой результат ждать.

Как использовать репетитору по китайскому:

Полезно подбирать учебную механику под конкретного ученика, а не снова назначать всем один и тот же режим.

Как использовать ученику:

Помогает выбрать рабочий способ учить китайский без суеты, мифов и бессмысленных марафонов.
