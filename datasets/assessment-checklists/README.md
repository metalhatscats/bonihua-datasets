# Диагностика уровня китайского — чек‑листы

Чек-листы оценки и диагностики по китайскому: прогресс, слабые места, готовность к следующему шагу.

Что это за набор:

Чек‑листы для оценки уровня и прогресса: что проверить, как оценить и какие признаки уровня считать ключевыми. Полезно преподавателям и ученикам перед сменой уровня или цели.

Как использовать репетитору по китайскому:

Полезно для стартовой диагностики, промежуточной оценки и честной обратной связи без воды.

Как использовать ученику:

Позволяет видеть прогресс по признакам, а не только по настроению после урока.
