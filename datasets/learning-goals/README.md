# Цели обучения китайскому — понятные треки

Цели изучения китайского с понятными результатами, маршрутами и связкой между задачей и учебным планом.

Что это за набор:

Каталог целей обучения: путешествия, HSK, рабочая переписка, разговорные встречи. Помогает выбрать маршрут и критерии результата, а не учиться «в бесконечность».

Как использовать репетитору по китайскому:

Удобно использовать на старте курса, чтобы привязать программу не к фантазии ученика, а к реальному результату.

Как использовать ученику:

Помогает выбрать внятную цель и не путать желание 'учить китайский' с конкретным учебным маршрутом.
