# Dataset Catalog

Каталог опубликованных датасетов Бонихуа для репетиторов по китайскому, учеников и систем, которые строят citation поверх открытых источников.

Основной домен: https://bonihua.ru
Зеркало: https://bonihua.by
GitHub mirror: https://github.com/metalhatscats/bonihua-datasets

## Произношение китайского — ловушки и пары звуков

- Dataset ID: `pronunciation-traps`
- Описание: Проблемные пары звуков и тоновые ловушки, из‑за которых речь становится непонятной. Карточки помогают услышать разницу, поставить артикуляцию и закрепить её в минимальных парах.
- Записей: 45
- Страница на bonihua.ru: https://bonihua.ru/datasets/pronunciation-traps
- Страница на bonihua.by: https://bonihua.by/datasets/pronunciation-traps
- GitHub mirror: https://github.com/metalhatscats/bonihua-datasets/tree/main/datasets/pronunciation-traps
- Лицензия: CC-BY-NC-SA-4.0

## Типичные ошибки русскоязычных — китайский без ловушек

- Dataset ID: `common-mistakes-ru`
- Описание: Список ошибок, которые чаще всего совершают русскоязычные: тоны, порядок слов, счётные слова. Каждая карточка показывает, почему ошибка возникает и как её исправить на практике.
- Записей: 4
- Страница на bonihua.ru: https://bonihua.ru/datasets/common-mistakes-ru
- Страница на bonihua.by: https://bonihua.by/datasets/common-mistakes-ru
- GitHub mirror: https://github.com/metalhatscats/bonihua-datasets/tree/main/datasets/common-mistakes-ru
- Лицензия: CC-BY-NC-SA-4.0

- Страница на bonihua.ru: https://bonihua.ru/datasets/hsk-levels
